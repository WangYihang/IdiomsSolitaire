# IdiomsSolitaire

一个成语接龙小脚本

#### API Usage

```bash
# Install it first
pip install IdiomsSolitaire
```

```bash
# Use it in your python source code
>>> import IdiomsSolitaire
>>> IdiomsSolitaire.init()
>>> print IdiomsSolitaire.guess("一心一意")
```


#### Script Usage

```bash
# Install it first
pip install -r requirements.txt
```
