# IdiomsSolitaire

A high-performance Chinese idioms solitaire tool.

## Installation

```bash
pip install -r requirements.txt
```

Or:

```bash
pip install .
```

## Quick Start

```bash
python IdiomsSolitaire.py 一心一意
```

Or if installed as a package:

```bash
idiomssolitaire 一心一意
```

### Custom Database

```bash
idiomssolitaire 一心一意 --db custom.db
```
